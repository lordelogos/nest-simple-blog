## Quick start

1. Clone the repo
2. Install deps

```bash
$ pnpm install
```

3. Setup Postgres locally (with Docker)

```bash
$ docker-compose up
```

4. Start the dev server locally

```bash
$ pnpm run start:dev
```

5. Open up swagger docs in your browser [localhost:3000/api](localhost:3000/api)
